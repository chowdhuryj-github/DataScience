Details about the dataset can be found at Kaggle. According to the dataset description:

  • Pclass is the Ticket Class. The best way to represent this feature is categorically, as it is a categorical feature representing the class of ticket using 1, 2 and 3.
  • SibSp is the Number of Sibling / Spouses. The best way to represent this is as a integer, as it isn't possible to have floating point values for a discrete feature.
  • Parch is the Number of Parents / Children. The best way to represent this is as a integer, as it isn't possible to have floating point values for a discrete feature.
  • Fare is the Passenger Fare. The best way to represent this is as a floating point value, as currency can have floating point values.
  • Cabin is a Number of the Cabin. The best way to represent this is as a integer, as it is a discrete numerical feature.
